DLL の作成と C のコンパイルのメモ
## ライブラリー
### ソースファイル
```c
// gcd.c
int gcd(int a, int b){
    return !b ? a : gcd(b, a % b);
}
```

### ヘッダーファイル
```c
// gcd.h
#ifndef TEST_H
#define TEST_H

int gcd(int a, int b);

#endif
```

### プログラム
```c
#include <stdio.h>
#include "gcd.h"

int main(void){
    printf("%d\n", gcd(24, 36));
}
```

## コンパイル

### DLL の作成
```sh
gcc gcd.c -shared -o gcd.dll
```

### コンパイル
```sh
gcc main.c -lgcd -L.
```
