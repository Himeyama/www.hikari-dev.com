添字を添えて for で繰り返し

for([i, e] of ["a", "b", "c"].entries()){
    console.log(i, e)
}

forEach でも

["a", "b", "c"].forEach((e, i) => {
    console.log(e, i)
})

長さ n の配列を作成

const n = 10
let ary = [...Array(n)].map((_, i) => i)
// もしくは
ary = Array.from({length: n}).map((_, i)=> i)

配列の和

let ary = [1, 2, 3, 4, 5]
ary.reduce((_, v) => _ + v)
