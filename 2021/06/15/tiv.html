ターミナル上で画像を表示する tiv コマンドの紹介。

ファイルエクスプローラー開くのがめんどくさいときに良い。

インストール方法

git clone https://github.com/stefanhaustein/TerminalImageViewer.git
cd TerminalImageViewer/src/main/cpp
make -j
sudo make install
